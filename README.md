# 🚂 سِكّة - مواعيد قطارات مصر | Sekka Egypt Train Schedules

![سِكّة - مواعيد قطارات مصر](https://raw.githubusercontent.com/y9090oussefwork/sekka-egypt-train-schedules/main/public/hero-train.png)

> **سِكّة (Sekka)** هو تطبيق ويب حديث، متكامل وسريع كلياً لمعرفة مواعيد قطارات هيئة سكة حديد مصر، والبحث في أكثر من **10,000 موعد وتوقف** عبر جميع المحطات في جمهورية مصر العربية مع دعم العمل بدون إنترنت (PWA) واكتشاف أسرع الرحلات وأقربها موعداً.

## ✨ المميزات الرئيسية (Key Features)

1. 🔍 **البحث الذكي بين المحطات**:
   - البحث عن القطارات المتاحة بين أي محطتين في مصر (مثل: القاهرة ← الإسكندرية، القاهرة ← أسوان).
   - إمكانية التبديل السريع بين محطة القيام ومحطة الوصول بنقرة واحدة.

2. 🚆 **البحث برقم القطار**:
   - الاستعلام المباشر عن خط سير أي قطار محدد ومعرفة جميع توقفاته ومواعيد الوصول والتحرك.

3. 📍 **لوحة حركات المحطة (Station Departure Board)**:
   - عرض جميع القطارات التي تمر بالمحطة المختارة في وقت معين مع التفاصيل.

4. ⚡ **الفلاتر والترتيب المتقدم**:
   - تصفية الرحلات حسب الوقت (صباحاً، ظهراً، مساءً، ليلاً).
   - تصفية حسب فئة القطار (مكيفة، مكيف روسي، VIP، تالجو، تحيا مصر، إلخ).
   - ترتيب النتائج حسب: **الأقرب موعداً**، **الأسرع وصولاً**، **الأقل توقفاً**، أو **رقم القطار**.

5. ⚖️ **مقارنة القطارات (Train Comparison)**:
   - تحديد حتى 3 قطارات ومقارنتها مباشرة من حيث وقت القيام، الوصول، مدة الرحلة، وعدد المحطات البينية.

6. ❤️ **المفضلة والسجل (Favorites & Recent Searches)**:
   - حفظ القطارات المتكررة في المفضلة للوصول السريع إليها.
   - حفظ عمليات البحث الأخيرة محلياً في المتصفح.

7. 🌙 **الدعم المزدوج للمظهر (Dark & Light Mode)**:
   - تصميم عصري يدعم الوضع الليلي والنهاري تلقائياً مع زر للتحويل.

8. 📱 **دعم العمل بدون إنترنت (Progressive Web App - PWA)**:
   - يتم تخزين قاعدة البيانات والمكونات محلياً عند الزيارة الأولى ليعمل التطبيق بكفاءة حتى عند ضعف الشبكة أثناء السفر.

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **الواجهة الأمامية**: [Next.js 16 / Vinext](https://github.com/cloudflare/vinext) + React 19 + TypeScript.
- **التنسيق والأنماط**: CSS3 مع متغيرات وتصميم متجاوب كامل مع كافة الشاشات (Responsive Mobile-First Design).
- **قاعدة البيانات محلياً**: قاعدة بيانات JSON شاملة تحتوي على جميع القطارات والمحطات والتوقفات في مصر (`public/data/train_data.json`).
- **أدوات البناء والتشغيل**: Vite 8 / Vinext + Cloudflare Workers / Vercel.

## 📥 شرح التثبيت والتشغيل المحلّي (Local Setup Guide)

### المتطلبات الأساسية (Prerequisites)
- تثبيت **Node.js** إصدار `>= 20.0.0` أو أعلى.
- تثبيت **npm** أو **yarn** أو **pnpm**.

### خطوات التشغيل:

1. **استنساخ المستودع (Clone Repository)**:
   ```bash
   git clone https://github.com/y9090oussefwork/sekka-egypt-train-schedules.git
   cd sekka-egypt-train-schedules
   ```

2. **تثبيت حزم التبعيات (Install Dependencies)**:
   ```bash
   npm install
   ```

3. **تشغيل خادم التطوير (Run Development Server)**:
   ```bash
   npm run dev
   ```
   افتح المتصفح على العنوان: `http://localhost:5173`

4. **بناء النسخة الإنتاجية (Production Build)**:
   ```bash
   npm run build
   ```

## 🚀 شرح النشر (Deployment Guide)

### 1. النشر على Vercel (موصى به)
يمكن نشر المشروع بسهولة وبشكل مجاني على Vercel:
```bash
# باستخدام Vercel CLI
npx vercel --prod
```
أو ربط حساب GitHub بموقع Vercel واختيار المستودع `sekka-egypt-train-schedules` وسيتم النشر تلقائياً عند كل Commit.

### 2. النشر على Cloudflare Pages
المشروع مبني ومتوافق كلياً مع Cloudflare Workers / Pages:
```bash
npx wrangler pages deploy dist
```

### 3. النشر على Netlify
يمكن ربطه أيضاً بـ Netlify وتحديد أمر البناء: `npm run build` ومجلد المخرجات `dist`.
